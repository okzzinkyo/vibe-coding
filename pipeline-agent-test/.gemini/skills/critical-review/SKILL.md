# 비판적 검토 스킬

## 과장 탐지 체크리스트
아래 표현이 있으면 근거 존재 여부를 확인:

| 표현 | 필요한 근거 |
|------|-------------|
| significantly | 통계 검정 (p-value) |
| outperforms | 동일 조건 비교 실험 |
| state-of-the-art | 최신 벤치마크 결과 |
| novel | 기존 연구와의 차이 명시 |
| first | "to the best of our knowledge" 한정어 |
| dramatically | 정량적 수치 |

## 인용 검증 절차
1. 초안에서 (Author, Year) 형태의 인용 모두 추출
2. research_plan.md의 논문 분석 카드와 대조
3. 주장과 인용된 논문의 실제 내용이 일치하는지 확인
4. 불일치 시 web_search로 원 논문 재확인

## 리뷰 분류 기준
- 🔴 Critical: 사실 오류, 인용 불일치, 논리적 모순
- 🟡 Major: 과장 표현, 불명확한 설명, 구조 문제
- 🟢 Minor: 문법, 표현 어색함, 서식

## 최소 검출 요건
**최소 3개 이상의 개선점을 반드시 찾아야 한다.**
완벽해 보여도 더 나아질 수 있는 부분은 항상 있다.
