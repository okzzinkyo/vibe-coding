# 연구 계획 수립 스킬

## 검색 전략
1. 핵심 키워드 추출 → 동의어 확장 → Boolean 조합
2. 검색 우선순위: Google Scholar → arXiv → Semantic Scholar
3. 선별 기준: 최근 3년, 피어리뷰, 인용 수 10+

## 논문 분석 카드 템플릿
```
## [번호] 논문 제목 (원어)
- 저자/학회/연도:
- 핵심 방법: (2문장)
- 주요 결과: (정량 수치 포함)
- 한계점:
- 우리 연구와의 관련성:
```

## Introduction 구조 설계 — 깔때기 원칙
```
문단 1: 큰 그림 (이 분야가 왜 중요한가)
문단 2: 구체적 문제 (어떤 한계가 있는가)
문단 3: 기존 시도 (다른 사람들은 어떻게 했는가)
문단 4: 우리 접근 (왜 다른가, 어떻게 해결하는가)
문단 5: 기여 + 예비 결과 (무엇을 보여줄 것인가)
문단 6: 논문 구조 안내 (optional)
```

각 문단은 다음을 명시해야 한다:
- 핵심 메시지 (한 문장)
- 뒷받침 근거 (어떤 논문/데이터)
- 다음 문단으로의 전환 논리
