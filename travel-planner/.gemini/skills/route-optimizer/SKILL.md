# Route Optimizer — 여행 동선 최적화 도구

일별 방문지 동선 최적화, 교통수단 선택, 시간 블록 배분 전략을 제공합니다.

## 핵심 원칙

### 5대 원칙
1. **클러스터링**: 인접 관광지를 같은 날에 묶는다.
2. **순환 경로**: 숙소 → 관광 → 식사 → 숙소 흐름을 우선한다.
3. **시간 정합**: 운영시간·휴무일·예약시간을 반영한다.
4. **피크 회피**: 인기 관광지는 오전 이른 시간이나 저녁에 배치한다.
5. **여유 확보**: 총 이동시간은 일일 2시간 안팎을 권장한다.

## 동선 설계 알고리즘
1. 모든 방문지 위치를 지역별로 묶는다.
2. 도보 15분 내 후보를 같은 클러스터로 본다.
3. 클러스터별 체류시간을 계산한다.
4. 일별 총 소요시간이 8~10시간을 넘지 않도록 나눈다.
5. 교통수단은 비용·정시성·환승 수를 함께 고려한다.
6. 식사/휴식 시간을 반드시 삽입한다.

## 시간 블록 템플릿
- 08:00–09:00 아침 + 이동
- 09:00–11:30 오전 핵심 관광
- 11:30–13:00 점심 + 여유
- 13:00–15:30 오후 관광 1
- 15:30–16:00 카페 휴식
- 16:00–17:30 오후 관광 2
- 17:30–18:30 이동 + 정리
- 18:30–20:00 저녁
- 20:00–21:30 야간 활동(선택)

## 교통수단 선택 힌트

### 도시 내
- 1km 이내: 도보 우선
- 3km 이상 / 정시성 중요: 지하철·전철 우선
- 환승이 많고 인원이 3명 이상: 택시 비교
- 비도심/근교: 버스·렌터카 여부 검토

### 도시 간
- 100~500km: 고속철 / 신칸센 우선 검토
- 경제성 중시: 버스 비교
- 섬 / 장거리: 항공 / 페리 검토

## 일정표에 반영할 최소 정보
- 이동 횟수
- 총 이동시간
- 대중교통 / 도보 / 택시 사용 구간
- 우천 시 대안
- 피로도 높은 구간 메모
