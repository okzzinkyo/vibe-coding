# SQL Daily

매일 오전 8시에 SQL 연습 문제를 이메일로 발송하는 자동화 스크립트.

## 동작 방식

- 8개 토픽(SELECT/WHERE, JOIN, 윈도우 함수 등)을 순환하며 문제 생성
- Claude API로 문제 생성 → 품질 리뷰 → MySQL 실행으로 예상 출력 검증
- 통과 시 Gmail SMTP로 구독자에게 발송

## 환경 변수

| 변수 | 설명 |
|---|---|
| `ANTHROPIC_API_KEY` | Claude API 키 |
| `GMAIL_USER` | 발신 Gmail 주소 |
| `GMAIL_APP_PASSWORD` | Gmail 앱 비밀번호 |
| `RECIPIENT_EMAILS` | 수신자 이메일 (쉼표 구분) |
| `TEST_EMAIL` | 테스트 발송용 단일 이메일 |

## 실행

```bash
node send.js
```

## 한계 및 향후 개발 필요 항목

### 개인화 난이도 미지원

현재 모든 구독자에게 동일한 문제를 발송한다. 이상적으로는 사용자별 SQL 실력에 맞는 레벨의 문제를 제공해야 하지만, 이메일 단방향 발송 구조상 사용자 레벨을 수집하거나 반영할 수단이 없다.

향후 웹 인터페이스를 도입할 경우, 가입 시 레벨 선택(초급/중급/고급)을 받아 개인화된 문제를 발송하는 방향으로 확장 검토.
